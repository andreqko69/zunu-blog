1) Використовуючи prompt (https://developer.mozilla.org/en-US/docs/Web/API/Window/prompt) запитати у користувача його вік. Значенням по замовчуванню вкажіть 18.
Якщо вік менше 18, вивести в консоль повідомлення про те, що доступ заборонено. Якщо вік 18 або більше, вивести повідомлення про те, що доступ дозволено
2) Використовуючи prompt запитати у користувача його імʼя та вік (двома окремими запитами). Дані зберегти в обʼєкт. Після цього вивести в консоль повідомлення типу: "Імʼя: <name>, вік: <age>"
3) Використовуючи prompt запитати у користувача 3 числа (окремим запитом для кожного). Вивести в консоль суму цих чисел.
4) Використовуючи prompt запитати у користувача 3 слова (окремим запитом для кожного). Обʼєднайте всі слова через пробіл і виведіть у консоль.

Note: Зверніть увагу, який тип даних повертає prompt. Якщо потрібно, зробіть перетворення типів. В посиланні на документацію
є інформація про те, який тип він повертає. Також, для виведення значень на екран можете використовувати alert() (https://developer.mozilla.org/en-US/docs/Web/API/Window/alert)
